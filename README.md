# Tech Stack

**Client:** NextJs, TailwindCSS, Hardhat, MetaMask

**Server:** NodeJs, Mysql

**Blockchain:** Etherscan

# Installation

Install front-end:

```bash
  npm install
  npx hardhat compile
  npx hardhat run --network testnet scripts/deploy.js
  npm run dev
```

Install back-end: 

```bash
  npm install
  npm start
```
